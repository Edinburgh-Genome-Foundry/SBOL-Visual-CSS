Drag&Drop editor

This example demonstrates a drag-and-drop editor using the library SBOL Visual CSS for SBOL symbols visualization. The drag-and-drop mechanism is powered by the Dragula library

Usage

Drag the genetic parts of the inventory into the construct box. The parts of the construct can still be changed place. Drag a part out of the construct box to delete it.
