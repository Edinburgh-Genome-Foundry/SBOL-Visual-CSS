# Visual SBOL CSS This library provides provides CSS classes to easily display schematic DNA sequences in the Visual SBOL standard: <div class="sbol-visual centered"> <div class="sbolv promoter"></div> <div class="sbolv cds"></div> <div class="sbolv terminator"></div> </div> ### Usage Download the source and place the `sbol-visual-css` folder in your project and link the css to your html document: ``` <link rel="stylesheet" href="sbol-visual-css/sbol-visual.css"></link> ``` Then in the body write a `sbol-visual` div with different `sbolvs`: ``` <div class="sbol-visual"> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds"><em>acs</em></div> <div class="sbolv terminator"></div> <div class="sbolv insulator"></div> </div> ``` They will be displayed as follows: <div class="sbol-visual"> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds"><em>acs</em></div> <div class="sbolv terminator"></div> <div class="sbolv insulator"></div> </div> ### Auto-wrapping When the number of `sbolv` elements increases, they are nicely wrapped to the line: <div class="sbol-visual"> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds"><em>acs</em></div> <div class="sbolv terminator"></div> <div class="sbolv insulator"></div> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds"><em>acs</em></div> <div class="sbolv terminator"></div> <div class="sbolv insulator"></div> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds"><em>acs</em></div> <div class="sbolv terminator"></div> <div class="sbolv insulator"></div> </div> ### Adaptive width Put any html you want into the `sbolv` divs, the SBOL symbols will stretch to make space: ``` <div class="sbol-visual"> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds">some longer gene name</div> <div class="sbolv terminator"></div> </div> ``` <div class="sbol-visual"> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds">some longer gene name</div> <div class="sbolv terminator"></div> </div> If the caption is longer than `sbolv.max-width` (default is `200px`) the caption will be wrapped: <div class="sbol-visual"> <div class="sbolv promoter"> P1 </div> <div class="sbolv ribosome-entry-site"> rbs 1 </div> <div class="sbolv cds">some really very long caption that you could have made shorter</div> <div class="sbolv terminator"></div> </div> ### Centering the symbols An easy way to center the symbols is to add `centered` to the `sbol-visual` div: ``` <div class="sbol-visual centered"> <div class="sbolv promoter"> P1 </div> <div class="sbolv cds">bla</div> <div class="sbolv terminator">t2</div> </div> ``` <div class="sbol-visual centered"> <div class="sbolv promoter"> P1 </div> <div class="sbolv cds">bla</div> <div class="sbolv terminator">t2</div> </div> Beware that all lines will be centered, even if they contain only a single symbol. ### Changing the symbol sizes Since the symbols are vector graphics they will render well at any size. The size of the symbols is driven by the height of the `sbolv` divs. To get smaller symbols you would first add CSS class `small` to `sbol-visual`: ``` .sbol-visual.small .sbolv { height: 80px; /* default is 120px */ min-width: 35px; /* default is 50px */ } ``` Then in hour html you add `small` to your div's classes: ``` <div class="sbol-visual small centered"> <div class="sbolv promoter"> P1 </div> <div class="sbolv cds">bla</div> <div class="sbolv terminator">t2</div> </div> ``` <div class="sbol-visual centered small"> <div class="sbolv promoter"> P1 </div> <div class="sbolv cds">bla</div> <div class="sbolv terminator">t2</div> </div> <style> .sbol-visual.small .sbolv { height: 80px; min-width: 35px; } </style> ### Notes For the moment there is no easy way to change the color of the symbols - apart from modifying the SVG files ### Contribute SBOL-visual ### Reference The supported classes for `sbolv` are in alphabetical order: - `blunt-restriction-site` - `cds` - `five-prime-overhang` - `five-prime-sticky-restriction-site` - `insulator` - `primer-binding-site` - `origin-of-replication` - `operator` - `promoter` - `protease-site` - `protein-stability-element` - `restriction-enzyme-recognition-site` - `ribonuclease-site` - `ribosome-entry-site` - `rna-stability-element` - `signature` - `terminator` - `three-prime-overhang` - `three-prime-sticky-restriction-site` - `user-defined` <a href="https://github.com/zulko"> <img style="position: absolute; top: 0; right: 0; border: 0; z-index: 1000; margin: 0;" src="https://s3.amazonaws.com/github/ribbons/forkme_right_darkblue_121621.png" alt="Fork me on GitHub"> </a>
