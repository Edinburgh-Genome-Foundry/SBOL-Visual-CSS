# Visual SBOL CSS This library provides provides CSS classes to easily display schematic DNA sequences in the Visual SBOL standard. Unzip the `visual-sbol-css` in your project and link the css to your html document: ``` <link rel="stylesheet" href="visual-sbol-css/visual-sbol.css"></link> ``` Then in the body write a `sbol-visual` div with different `sbol-elements`: ``` <div class="sbol-visual"> <div class="sbol-element promoter"> P1 </div> <div class="sbol-element ribosome-entry-site"> rbs 1 </div> <div class="sbol-element cds"><em>acs</em></div> <div class="sbol-element terminator"></div> <div class="sbol-element insulator"></div> </div> ``` They will be displayed as follows: <div class="sbol-visual"> <div class="sbol-element promoter"> P1 </div> <div class="sbol-element ribosome-entry-site"> rbs 1 </div> <div class="sbol-element cds"><em>acs</em></div> <div class="sbol-element terminator"></div> <div
