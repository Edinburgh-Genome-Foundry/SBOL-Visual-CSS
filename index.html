# Visual SBOL CSS This library provides provides CSS classes to easily display schematic DNA sequences in the Visual SBOL standard. For instance it transforms this: ``` <div class="sbol-visual"> <div class="sbol-element promoter"> P1 </div> <div class="sbol-element ribosome-entry-site"> rbs 1 </div> <div class="sbol-element cds"><em>acs</em></div> <div class="sbol-element terminator"></div> </div> ``` into this: <div class="sbol-visual"> <div class="sbol-element promoter"> P1 </div> <div class="sbol-element
