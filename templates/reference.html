    Modifier classes

    These classes can be combined with sbolv or sbol-visual to alter rendering behaviour.

    Class Applies to Effect
    centeredsbol-visualCenters wrapped lines within the container.
    inlinesbolv, sbol-visualRenders the symbol(s) inline, vertically aligned with surrounding text.
    antisensesbolv, sbol-visual, inner label elementMirrors the element horizontally. Apply to a child text node to keep the label readable.
    invertedsbolv, sbol-visualInverts symbol colours for use on dark backgrounds.
    combinatorialwrapper div inside sbol-visualGroups alternative symbol options for one design slot.
